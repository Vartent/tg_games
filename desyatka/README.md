# Десятка (Ten!)

Скоростной пазл-змейка «собери цепочку с суммой 10». Telegram Mini App. Полная спека — [docs/SPEC.md](docs/SPEC.md).

## Варианты (A/B)

Целевая сумма — сборочный параметр `VITE_CHAIN_TARGET` (по умолчанию `10`):

- **Десятка** (этот каталог): `VITE_CHAIN_TARGET=10` — цепочки ровно 10 или 20.
- **Дюжина** (`../dyuzhina`): `VITE_CHAIN_TARGET=12` — цепочки ровно 12 или 24, бот @even_dozen_bot.

Название игры, подписи («дес.»/«дюж.»), шер-тексты и ключ профиля подстраиваются автоматически.
Правки геймплея делаются здесь и переносятся в `../dyuzhina` копированием `src/` и `tests/`.
Юнит-тесты пинуют дефолтный вариант (10).

## Структура

```
src/types.ts      — типы (готово)
src/constants.ts  — правила в числах (готово)
src/game.ts       — логика, чистые функции (СТАБЫ — реализовать по TDD)
tests/            — красные тесты, фиксируют спеку
```

## TDD-процесс

```bash
npm install
npm test          # сейчас всё красное — это техзадание
npx tsc --noEmit  # типы должны проходить всегда
```

Порядок имплементации:

1. `tests/board.test.ts` — генерация доски: детерминизм, диапазон цифр, гарантия хода.
2. `tests/moves.test.ts` — валидация прямоугольника, применение хода, очки.
3. `tests/round.test.ts` — таймер раунда, rewarded-продление, конец игры.
4. `tests/daily.test.ts` — seed дня, одна попытка, стрик, шер-карточка.

После зелёных тестов: DOM-UI (сетка, тач-выделение, конфетти), Telegram SDK (initData, CloudStorage), Adsgram (rewarded + interstitial), лидерборды, бот-пуши.

## Правила для game.ts

- Чистые функции, без мутаций. Никаких `Date.now()` / `Math.random()` — время и seed параметрами.
- PRNG — детерминированный (mulberry32 или аналог), сидируемый числом.
- Ошибки правил — `throw new GameError(code)`.

## Definition of done (MVP)

- [x] Все тесты зелёные (38), `tsc --noEmit` чистый.
- [x] UI: доска, тач-прямоугольник, лопанье + конфетти, таймер, экран результата с шер-кнопкой.
- [x] Дейли-режим (одна попытка, общая доска дня) + бесконечный, стрик.
- [x] Telegram SDK: ready/expand, CloudStorage с фолбэком на localStorage, хаптика, шер в чаты.
- [x] Adsgram-адаптер: rewarded +30 сек, interstitial раз в 2 раунда бесконечного (без блок-id работает dev-стаб).
- [ ] Лидерборды (дневной/вечный) — нужен backend.
- [ ] Бот-пуш «новая доска» — нужен бот-токен и backend (cron).
- [ ] Прогон на iOS/Android Telegram — нужен задеплоенный HTTPS-URL и бот.

## Запуск

```bash
npm run dev       # локально в браузере
npm run build     # прод-сборка в dist/
```

## Что нужно для боевого запуска

1. Задеплоить `dist/` на HTTPS (Cloudflare Pages / Vercel).
2. Создать бота у @BotFather → Mini App URL.
3. Зарегистрироваться в Adsgram, получить block id → переменная `VITE_ADSGRAM_BLOCK_ID`.
4. `VITE_BOT_URL=https://t.me/<bot>` — для шер-ссылок.
5. Backend для лидербордов и пушей (можно после теста трафика).
