# Трио (TenThirds)

«Три в ряд» на сумму: свапни две соседние клетки так, чтобы строка или колонка
дала отрезок с суммой ровно 10 — он лопается. Каскады без ограничений: гравитация
сводит цифры, новые десятки лопаются сами. Telegram Mini App.
Бот: [@ten_thirds_bot](https://t.me/ten_thirds_bot).

## Правила

- Ход — свап двух соседних по стороне клеток свайпом. Если ничего не лопается —
  свап не происходит, штраф временем (1с, при спаме 2с, 3с...).
- Лопаются ВСЕ горизонтальные/вертикальные отрезки с суммой ровно 10 (длина от 2).
  Лимита на количество нет — хоть всё поле.
- 1 отрезок = 1 зачёт = 10 очков. 2+ отрезков за ход (с каскадами) — огонёк ×2 на 10с
  и +10с к таймеру.
- Зеро (зелёный 0, с 8-го уровня): сумму не меняет, лопнувший отрезок с зеро даёт ×2
  этому же ходу и зажигает огонёк.
- Уровни, цели, время, гемы и продолжение — как в «Десятке» (общая прогрессия).

## Ключевое инженерное решение

Досыпка и генерация поля **никогда сами не замыкают десятки** (конструктивный подбор
номиналов): иначе при правиле «сумма 10» случайная досыпка почти всегда создаёт новые
отрезки и каскад не гаснет (проверено — лавина до предохранителя CASCADE_CAP).
Каскады создаёт только гравитация, сводящая выжившие цифры.

Если после хода на поле нет ни одного валидного свапа — поле перемешивается заново.

## Структура и запуск

Движок — чистые функции в `src/game.ts` (без Date.now/Math.random), тесты в `tests/`.

```bash
npm install
npm test             # 38 тестов движка
npm run dev
./scripts/deploy.sh  # сборка + выкладка на trader-vps
```
