# Addictive Games — линейка Telegram Mini App игр

Цель: линейка простых, визуально сочных игр-таймкиллеров в Telegram Mini Apps с монетизацией через рекламу (rewarded video, interstitial) и закупкой трафика через Telegram Ads.

## Проекты

| Директория | Игра | Жанр | Статус |
|---|---|---|---|
| `oil-fleet/` | **Флотилия** — merge-idle нефтяных вышек на сетке океана | merge-idle | спека + красные тесты, ждёт разработки |
| `desyatka/` | **Десятка** — пазл-змейка «собери цепочку с суммой 10/20» | скоростной пазл | **в проде**: https://207-148-119-174.sslip.io/ (бот @tennishgame_bot) |
| `dyuzhina/` | **Дюжина** — A/B-вариант Десятки с целью 12/24 (тот же движок) | скоростной пазл | **в проде**: https://dyuzhina.207-148-119-174.sslip.io/ (бот @even_dozen_bot) |

`docs/` в корне — архив ранних спек (спека 1 «Глубина» отклонена, спека 2 устарела — актуальная версия в `oil-fleet/docs/SPEC.md`).

## Порядок запуска линейки

1. **Десятка** — разведчик: самый быстрый MVP, тестирует воронку Telegram Ads (CTR/CPC/D1) за ~$300.
2. **Флотилия** — основной продукт с длинной петлёй ретеншна.
3. Кросс-промо между играми — начало внутренней сети трафика.

## Процесс разработки

Обе игры разрабатываются по TDD: в `tests/` лежат красные тесты, фиксирующие всю игровую логику из спеки. Definition of done и порядок имплементации — в README каждой игры.

```bash
cd oil-fleet && npm install && npm test   # красные — это ожидаемо до имплементации
cd desyatka && npm install && npm test
```
