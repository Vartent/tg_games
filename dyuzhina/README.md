# Дюжина (Dozen!)

Скоростной пазл-змейка «собери цепочку с суммой 12». A/B-вариант «Десятки» — тот же движок,
другая целевая сумма. Бот: [@even_dozen_bot](https://t.me/even_dozen_bot).

## Связь с «Десяткой»

Кодовая база идентична `../desyatka` — вариант задаётся сборочным параметром
`VITE_CHAIN_TARGET=12` (зашит в скрипты `dev`/`build` в package.json). Название игры,
подписи («дюж.»), шер-тексты и ключ профиля (`chain12_profile_v1`) подстраиваются автоматически.
Цепочки — ровно 12 или 24, потолок выделения 24, максимум 2 зачёта с цепи; очки = зачёты × 12.

**Синхронизация:** правки геймплея делаются в `desyatka/` и переносятся сюда копированием
`src/` и `tests/` (расходятся только package.json, index.html, README, scripts/deploy.sh).

Юнит-тесты гоняют общий движок в дефолтной конфигурации (цель 10) — математика
параметризована и одинакова для обоих вариантов.

## Запуск

```bash
npm install
npm run dev        # локально, уже с целью 12
npm test           # 59 тестов движка
./scripts/deploy.sh  # сборка + выкладка на trader-vps
```
